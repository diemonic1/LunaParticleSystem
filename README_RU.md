# Luna Particles Tuner
Плагин для быстрого редактирования настроек Particle System в билде для Luna Playworks

# Установка
1) Скачайте последнюю версию плагина на [странице релизов](https://github.com/diemonic1/LunaParticleSystem/releases/latest)

В релиз входит zip архив `LunaParticlesTunerWebExtension.zip` с расширением для браузеров на базе chromium и пакет `LunaParticlesTuner.unitypackage`

2) - Браузерное расширение необходимо установить в ваш браузер.
   - Распакуйте архив в любое удобное место. Папку с расширением нельзя удалять — браузер читает расширение прямо из неё.
   - Откройте `chrome://extensions` (для Edge — `edge://extensions`).
   - В правом верхнем углу включите тумблер Developer mode.
   - Нажмите Load unpacked и выберите папку, в которую распаковали архив.

3) - Пакет `LunaParticlesTuner.unitypackage` необходимо добавить в ваш проект.
   - Откройте в Unity папку Assets в разделе Project, нажмите правую клавишу мыши и выберите Import Package - Custom Package, а затем выберите `LunaParticlesTuner.unitypackage`
   - импортируйте ассеты плагина в проект.

# Использование
> [!CAUTION]
> После настройки ваших частиц ОБЯЗАТЕЛЬНО уджалите плагин из проекта, чтобы исключить попадание его кода в релизную версию плеебла. Если этого не сделать, плагин может значительно ухудшать перфоманс плеебла во время его работы.

1) После установки пакета в Unity и расширения в браузере выберите любую Particle System на сцене и добавьте к ней компонент `LunaParticlesTuner`.
После добавления убедитесь, что поле Particle System ссылается на систему партиклей на этом объекте.
Вы можете добавлять сколько угодно систем частиц для их дальнейшей настройки.

<img width="502" height="393" src="https://github.com/user-attachments/assets/dc1bd0f9-f974-401e-a4cb-f44616148718" />

2) Соберите Develop версию билда через Tools - Unity Playworks Plugin - собрать Develop.

> [!CAUTION]
> Система частиц появится в настройках только после того, как GameObject, на котором она находиться, будет включен. Если он включен на сцене изначально - настройка появится сразу. Чтобы иметь возможность настроить систему частиц заранее, оставьте GameObject включенным, и вручную запускайте частицы с помощью _particleSystem.Play(); 

3) После успешной сборки билда откройте его в браузере. Когда билд загрузиться, внизу страницы появиться кнопка открытия настроек систем партиклей. При ее нажатии открывается боковое меню.

4) В боковом меню вы можете:
   - Включать отладочные логи плагина (пишутся в консоли инструментов разработчика - f12).
   - Изменять TimeScale внутри билда (например, чтобы рассмотреть частицы).
   - Изменять любые предоставленные настройки систем частиц, чтобы настроить их вид и поведение. Обратите внимание, что некоторые настройки вовсе недоступны для изменения, а у некоторых свойств, у которых в качестве источников данных можно выбрать константные значения или кривые, невозможно изменять параметры при выборе кривых.

5) После настройки системы частиц необязательно переносить отдельные значения каждого измененного поля вручную. У каждой системы частиц есть кнопка копирования настроек. При ее нажатии в буфер обмена копируется JSON строка со всем настройками. Вы можете вставить ее в соответствующее поле на компоненте `LunaParticlesTuner`, и после нажатия кнопки Apply JSON все параметры применятся к системе частиц автоматически.
